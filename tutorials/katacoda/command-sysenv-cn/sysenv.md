
`sysenv`{{execute T2}} 命令可以查看当前JVM的环境属性(`System Property`)，与`sysprop`类似

## 使用参考

`sysenv -h`{{execute T2}} 

```
 USAGE:
   sysenv [-h] [env-name]

 SUMMARY:
   Display the system env.

 EXAMPLES:
   sysenv
   sysenv USER

 WIKI:
   https://arthas.aliyun.com/doc/sysenv

 OPTIONS:
 -h, --help                                                 this help
 <env-name>                                                 env name
```
