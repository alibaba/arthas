查看当前线程信息，查看线程的堆栈

### 参数说明

| 参数名称    | 参数说明                                |
| ----------- | --------------------------------------- |
| id          | 线程 id                                 |
| [n:]        | 指定最忙的前 N 个线程并打印堆栈         |
| [b]         | 找出当前阻塞其他线程的线程              |
| [i <value>] | 指定 cpu 占比统计的采样间隔，单位为毫秒 |
| [--all]     | 显示所有匹配的线程                      |
