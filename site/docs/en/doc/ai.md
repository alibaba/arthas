# AI Integration

Arthas can integrate with AI assistants through MCP Server, allowing AI clients to execute Arthas diagnostic commands as tool calls. This is useful for Java troubleshooting, runtime analysis, and automated diagnostics.

## MCP Server

- [Arthas MCP Server](mcp-server.md): Learn how to enable the MCP service, configure AI clients, use Streamable HTTP transport, and view supported diagnostic tools.
