session
===

查看当前会话的信息，显示当前绑定的pid以及会话id。

> 如果配置了tunnel server，会追加打印 代理id、tunnel服务器的url以及连接状态。
>
> 如果使用了staturl做统计，会追加显示statUrl地址。



### 使用参考

```
$ session
  Name        Value
--------------------------------------------------
 JAVA_PID    14584
 SESSION_ID  c2073d3b-443a-4a9b-9249-0c5d24a5756c
 
```
