# session

查看当前会话的信息，显示当前绑定的 pid 以及会话 id。

提示

如果配置了 tunnel server，会追加打印 代理 id、tunnel 服务器的 url 以及连接状态。

如果使用了 staturl 做统计，会追加显示 statUrl 地址。

# 使用参考

$ session
  Name        Value
--------------------------------------------------
 JAVA_PID    14584
 SESSION_ID  c2073d3b-443a-4a9b-9249-0c5d24a5756c
